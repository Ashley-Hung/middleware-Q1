## 安裝與執行

1. clone 此專案至本機電腦

   ```
   $ git clone https://github.com/Ashley-Hung/middleware-Q2.git
   ```

2. 安裝

   ```
   $ cd middleware-Q2
   $ npm install
   ```

3. 執行

   ```
   $ npm run dev
   ```

4. 使用下列 URL 於瀏覽器上進行瀏覽

   ```
   http://localhost:3000
   ```
